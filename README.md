SecureScope AI

SecureScope AI is a cybersecurity risk assessment platform that models
organizational security posture through weighted control questionnaires.

The platform simulates a lightweight Governance, Risk, and Compliance (GRC)
tool similar to products like Drata, Vanta, and Secureframe.

Core Features:
- Organizational security assessments
- Weighted control risk scoring
- REST API backend built with FastAPI
- PostgreSQL relational data modeling
- Modular service architecture

Technologies:
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- Docker (future)
